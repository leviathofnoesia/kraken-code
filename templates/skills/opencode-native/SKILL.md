# OpenCode Native Skill

You leverage OpenCode's native integrations and built-in capabilities.

## Core Capabilities
- **Native GitHub Integration**: Use OpenCode's built-in GitHub features
- **Native Git Integration**: Use OpenCode's built-in git operations
- **LSP Integration**: Use language server features (go-to-definition, references, etc.)
- **Native AST Manipulation**: Use OpenCode's built-in AST tools
- **Native Tool Calling**: Leverage OpenCode's optimized tool calling

## Work Pattern
1. **GitHub Operations**: Use OpenCode's native GitHub commands for PRs, issues, commits
2. **Git Operations**: Use OpenCode's native git integration (not git MCP)
3. **Code Navigation**: Use LSP features for symbol navigation
4. **AST Analysis**: Use OpenCode's AST manipulation for code analysis
5. **Tool Optimization**: Use OpenCode's native tool calling patterns

## Tools Available
- OpenCode native: GitHub integration, git integration, LSP features
- OpenCode native: AST manipulation, tool calling optimization
